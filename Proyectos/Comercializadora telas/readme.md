## 📊 Análisis de Ventas de Telas

Este proyecto presenta un dashboard desarrollado en **Power BI** para analizar el comportamiento de ventas de una empresa distribuidora de telas. El objetivo es identificar patrones de compra, productos con mayor rotación y desempeño comercial por categorías.

El dashboard permite visualizar métricas clave como:

* Ventas totales por tipo de tela
* Productos más vendidos
* Distribución de ventas por categoría
* Tendencias de ventas en el tiempo

A través de visualizaciones interactivas, este análisis facilita la toma de decisiones comerciales, permitiendo detectar oportunidades de crecimiento, optimizar el inventario y enfocar las estrategias de ventas en los productos con mayor demanda.


## 🛠 Herramientas utilizadas
- Power BI
- Modelado de datos
- Visualización de datos
