## 🏦 Análisis de Inteligencia Bancaria

Este proyecto presenta un dashboard desarrollado en Power BI enfocado en el análisis integral de un banco, abarcando el comportamiento de clientes, el portafolio de créditos, el riesgo financiero y la rentabilidad del negocio.

El objetivo principal es facilitar la toma de decisiones estratégicas mediante el análisis de datos financieros, permitiendo identificar oportunidades de crecimiento, controlar el riesgo y optimizar el desempeño del banco.

El dashboard permite visualizar métricas clave como:

Cartera total de créditos
Tasa de morosidad
Ingresos por productos financieros
Segmentación de clientes
Comportamiento de transacciones
Rentabilidad del banco


## 🛠 Herramientas utilizadas
- Power BI
- DAX (Data Analysis Expressions)
- Modelado de datos (modelo estrella)
- Visualización de datos
- Storytelling con datos

## 📈 Insights principales
- Los clientes del segmento Premium generan un mayor ingreso promedio para el banco.
- Algunos tipos de crédito concentran la mayor parte del portafolio y de los ingresos.
- Se identifican segmentos de clientes con mayor riesgo de morosidad.
- El crecimiento de ingresos muestra tendencias positivas en determinados periodos.
- Existen oportunidades de negocio en clientes con alto ingreso pero score medio.


## 📊 Estructura del dashboard

El proyecto está dividido en 5 páginas principales:
- Executive Overview → visión general del banco
- Comportamiento del Cliente → segmentación y análisis de clientes
- Análisis de Créditos → evaluación del portafolio crediticio
- Riesgo y Morosidad → identificación de riesgos financieros
- Rentabilidad → análisis de ingresos y valor del negocio
